Quick Start

Get running in
three commands

Uses the official palera1n v2.2.1 armel binary — statically linked, no compilation required. Works on any ARMv7 Linux device.

bash — postmarketOS ARMv7
# Download the binary from releases
$ curl -L https://github.com/noxbitx/s3ra1n/releases/latest/download/palera1n-linux-armel -o ~/palera1n
$ chmod +x ~/palera1n

# Kill usbmuxd — critical, don't skip this
$ sudo killall usbmuxd 2>/dev/null; sudo rm -f /var/run/usbmuxd

# First run: create fakefs (-B flag)
$ sudo ~/palera1n -f -B

# Every reboot after: just re-run without -B
$ sudo ~/palera1n -f
⚠ Charge your iPhone to at least 30% before starting. Battery death during PongoOS boot can corrupt the fakefs partition.
